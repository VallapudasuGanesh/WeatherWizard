
Weather App

This is a simple Python application that fetches weather data using the OpenWeatherMap API and displays it to the user.

Setup : 

1.Obtain API Key

Sign up on OpenWeatherMap to get an API key.

2.Insert API Key

Replace 'YOUR_API_KEY_HERE' in the code with your actual API key.

Usage :

1.Run the Python file weather_app.py.

2. Enter the name of the city you want to get weather data for.

3.The program will fetch and display the weather information.
